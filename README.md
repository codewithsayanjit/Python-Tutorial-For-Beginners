Before running the scripts, make sure you have Python installed on your local machine:
* Python 3.x or higher (Download from [python.org](https://www.python.org/downloads/))
* A code editor or IDE (e.g., VS Code, PyCharm, or IDLE)

## 📚 Topics Covered

* **Introduction to Python:** Setting up the environment and writing your first `hello_world.py`.
* **Variables & Data Types:** Understanding Integers, Floats, Strings, and Booleans.
* **Control Flow:** Working with `if-elif-else` statements, `for` loops, and `while` loops.
* **Data Structures:** Managing collections using Lists, Tuples, Dictionaries, and Sets.
* **Functions:** Creating reusable blocks of code, arguments, and return values.

## 💻 How to Run the Scripts

Open your terminal or command prompt, navigate to the specific folder containing the script, and run:

```bash
python script_name.py

```

*(Replace `script_name.py` with the actual name of the file you want to execute, e.g., `python hello_world.py`)*
